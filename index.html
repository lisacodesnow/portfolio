Guess the Word

Guess the Word is a word guessing game that requires players to guess letters, correctly or incorrectly, before they use their allotted guesses in order to win. This project uses vanilla Javascript to check the user’s input and match it with the correct word. The guessed words are fetched from a text file and used randomly.

Skills:

Vanilla Javascript, Manipulating the DOM, CSS, HTML5

Tools:

Git, Github,Dreamweaver

Live Page

The Name Tag Generator displays any name that a user adds. The name tags are created using JSX and stateful components. Lifecycle methods, mounting, unmounting, and updating, are also used to save a name tag once the screen is refreshed or closed then opened.

React, ES6, JSX, Managaing Data Flow
