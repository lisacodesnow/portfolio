Projects

The unplugged retreat project is a three page responsive website. A Photoshop design comp was used as well as flexbox for the layout. With media queries this site adapts to mobile, tablet, and desktop screens.

The graduate testimonial website is a single page displaying positive reviews from students. This project utilizes flexbox, grid, is responsive and is styled in various sizes. SCSS was helpful with assigning variables to frequently used colors to create a cohesive display and reduce syntax clutter.

The Github Repo Gallery is a project that pulls data from my Github portfolio site. A gallery of repositories are created using API’s to fetch the Github data and CSS. The repositories can be searched through and once clicked can take the user to that github’s repository.

Guess Word is a word guessing game that requires players to guess letters, correctly or incorrectly, before they use their allotted guesses in order to win. This project uses vanilla Javascript to check the user’s input and match it with the correct word. The guessed words are fetched from a text file and used randomly.

The Name Tag Generator displays any name that a user adds. The name tags are created using JSX and stateful components. Lifecycle methods, mounting, unmounting, and updating, are also used to save a name tag once the screen is refreshed or closed then opened.
