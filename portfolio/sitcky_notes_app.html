Sticky Notes App

Go to Live Page

Overview

The Super Sticky Notes app is an interactive app that allows users to create “sticky” notes on the computer to keep track of to-do items. React is using JSX to render the app's UI and mounting components to save the user’s items after they close the browser.

Challenge

The challenge I had was knowing how to edit certain fields of the note, either the title or the description.

Solution

The physical note is divided between two input sections: the title, and description.

But the actual makeup of the notes object has four key-value pairs that are needed for it to be created: id, title, description ,and doesMatchSearch.

The note's id is unique and is defined by the date of the note's creation. While it's a good way to find a note that needs to be edited, the id alone won’t allow me to edit the title and description field.

The solution to editing either or both the description and/or title is by utilizing keys. Keys like ids are assigned as unique identifiers to elements.

In this case, two keys are created, “title” and “description”. These keys along with the note’s id are sent to the editingNotes function that uses them to know if either title or description field needs to change.
