Unplugged Retreat

Go to Live Page

Overview

The unplugged retreat project is my first three-page responsive website that I coded. A Photoshop design comp and flexbox was used for the layout. With media queries, this site adapts to mobile, tablet, and desktop screens.

Challenge

A challenge I had was optimizing the website for mobile use.

Solution

When beginning this project, I was prepared to code the desktop version first and was worried about how I would convert all the content from a big desktop to a tiny, mobile screen.

One of the most popular ways users view the web is on their phones. So to optimize the website for mobile use, I coded the CSS mobile-first.

Mobile-first is when the website is coded to fit a mobile screen first, then the tablet and desktop versions are coded afterward.

It’s also a good way to see a simple version of the website before adding more code for tablet and desktop versions.

After the mobile version is coded, I added media queries for the tablet and desktop versions.

Starting small allowed me to fix any issues I was having with my HTML and CSS before scaling the website to tablet and desktop versions. Especially when creating multiple pages, this method made coding more manageable.

To understand the basics of what’s needed for a project is very helpful before over complicating the process and adding more sophisticated code.
