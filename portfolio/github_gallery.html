GitHub Repo Gallery

Go to Live Page

Overview

The GitHub Repo Gallery is a project that pulls data from my GitHub portfolio site. A gallery of repositories is created using APIs to fetch the GitHub data and CSS. The repositories can be searched through and once clicked can take the user to that GitHub’s repository.

Challenge

A challenge I had was displaying the UI from the APIs.

Solution

Using the async function and then passing the API results to another function that handles the UI is how I was able to display the UI from the APIs.

Before adding JavaScript, the UI only displays elements from the HTML document. The CSS that's needed is already written and will be added later by using the classList property in JavaScript.

Two UI sections that need APIs, the user profile’s information ,and a list of repositories.

For the user profile, the API is fetched with an async function. The results of the data are then passed to another function that handles the UI.

The data from the API is an object and the name, age, location, photo, and number of repositories are the properties used for the user profile.

For the list of repositories UI, the repositories API is fetched with a different async function. The data results are then passed to another function that handles the UI. The user's repositories are looped through and added as a list that runs the length of the webpage.
