Github Repo Gallery

Overview

The Github Repo Gallery is a project that pulls data from my Github portfolio site. A gallery of repositories are created using API’s to fetch the Github data and CSS. The repositories can be searched through and once clicked can take the user to that github’s repository.

Challenge

A challenge I had was displaying the UI from the API’s.

Solution

Using the async function and then passing the API results to another function that handles the UI is how I was able to display the UI from the API’s. Before adding Javascript, the UI only displays elements from the html document. The CSS that’s needed is already written and will be added later by using the classList property in Javascript.

There are two UI sections that need API’s, the user profile’s information and list of repositories.

For the user profile the API is fetched with an async function. The results of the data are then passed to another function that handles the UI. The data from the API is an object and the name, age, location, photo, and number of repositories are the properties used for the user profile.

For the list of repositories UI, the repositories API is fetched with a different async function. The data results are passed to another function that handles the UI. The users repositories is looped through and added as a list that runs the length of the webpage
