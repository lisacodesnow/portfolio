Overview

Guess the Word is a word guessing game that requires players to guess letters, correctly or incorrectly, before they use their allotted guesses in order to win. The guessed words are fetched from a text file and used randomly. This project uses vanilla JavaScript to check the userâ€™s input to match it with the correct word.

Challenge

This is one of my first JavaScript projects and knowing what to code and when can be confusing. The challenge I had was decomposing the project into sections for a clearer understanding on how to tackle the project.

Solution

Understanding how the game works and knowing what UI changes when the game is being played is how I was able to decompose the project into clearer sections.

Decomposing, or breaking down the UI that changes was a good starting point for me to know what to code for those changes. I then broke those sections further down into functions that work for the desired outcome of the game.

The first thing I did was analyze the UI that changes when a player inputs a letter.

When the letters inputted and the player hits the guess button the following parts change:

  • Message to user that displays if the correct letter was guessed
  • The word in progress which is the word that is being guessed
  • Remaining guesses that the player has to either win or lose
  • Guessed letters so the player won't input the same letter
  • Text input where the player enters the letter

And then once the game is over and regardless if you won or loss the following parts also change:

  • Congrats Message if the correct word is guessed
  • Play Again button appears

Tackling any project can be overwhelming, so having a strategy that outlines clear steps I need to take allows me to code a successful application.
