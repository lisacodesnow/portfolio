Sticky Note

Overview

The Note Taking app is an interactive interface that allows users to create notes that are displayed on the screen. React is using JSX for the apps UI and stateful components to manage the data typed by the user.

Challenge

The challenge I had was making the application interactive so it could display the user's typed words.

Solution

So far I had built applications with functional components that used an already built data model. In order for notes, the user’s typed words, to be received the state needed to have empty data models.

Utilizing event listeners and handlers is how the app became interactive and can display user’s typed words.

Event listeners listen for an event to happen i.e typing, and the event handler is triggered once the text is typed or button is hit. The event handler actually updates the state so the changes can be displayed on the screen.

Two event listeners and handlers were used:

  • The onChange listener is used with the input element and listens for a change in value, i.e when text is typed. Once the text is typed it calls the updateChalk event handler.

  • The onSubmit listener is used with the form element and triggers the updateNotes event handler when the submit button is pushed.
